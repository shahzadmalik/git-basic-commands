8 Basic Git Commands

1) git config --global user.email "emailofuser@email.com" git config --global user.name "username here"

2) git init

3) git add [filename] || -A "This command add files into Staging area which means these files are now ready to get committed"

4) git status

5) git commit -m "your message here"

6) git remote add origin repoName "This command just establish a link between local project with your repository but nothing is moved to remote yet"

7) git clone repoName

8) git pul origin master "This commands pull any files which might have been added in the master by other users but not reflecting in your local copy"
